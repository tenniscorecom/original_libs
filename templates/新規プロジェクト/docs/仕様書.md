## 6. 構成

```
main.py            エントリポイント（ログ初期化・エラー受け）
config.ini         設定（.gitignore。example をコピーして作る）
実行.bat           起動用（ダブルクリック）
認証情報の登録.bat  登録 GUI をダブルクリックで開く
src/
  config.py          設定シングルトン（config = Config()。補完スタブ config.pyi を自動生成）
  run.py             処理の本体
  credentials.py     使う認証情報の宣言
  browser_options.py ブラウザ設定（ブラウザを使う場合。使わなければ削除）
docs/
  使い方.md         操作手順（非エンジニア向け）
  仕様書.md         この文書
  ERRORS.md         エラー対応
```

comken は共有サーバーを直接参照する（`実行.bat` が起動時に `PYTHONPATH` を設定）。

---

## 7. 実行方法

- 通常: `実行.bat` をダブルクリック（または `python main.py`）
- 動作確認: `main.py` の `set_dry_run(True)` を有効にすると、保存・送信をスキップして流れだけ確認できる
